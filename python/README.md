
SETUP
-----

* Follow the setup instructions in the root.  In particular create a
  virtualenv and activate it.

* Install libygp (from ../libygp).

* Install libffi.  E.g., `sudo apt-get install libffi-dev` if on
  Debian or Ubuntu.

* Run

```sh
pip install -e .
python setup.py develop
```
